# 省级通用数据交换平台
###[平台简介]
    基于现有多个业务系统的数据和健康信息平台的数据，数据经统一的标准和要求进行采集整合后，可以以消费方申请（订阅）的形式，定制筛选数据子集，按日/周/月/季/年度时间维度、机构维度和数据元维度向第三方进行数据推送（发布）。资源发布方发布资源，资源申请方申请资源，定时数据交换。
